# Contributing

Thanks for helping improve the MSc AI directory!

## Adding or Updating a Course

All course data lives in `data/courses.json`. Each entry looks like this:

```json
{
  "name": "MSc Artificial Intelligence",
  "university": "University of Example",
  "slug": "example-artificial-intelligence",
  "url": "https://example.ac.uk/courses/ai-msc",
  "category": "Full Time MSc AI (UK)",
  "duration": "1 year",
  "fees_domestic": "£12,000",
  "fees_international": "£28,000",
  "entry_requirements": "2:1 in a relevant subject",
  "summary": "A one-year programme covering core AI topics.",
  "notes": ""
}
```

To add a course, append a new entry to the array. To update one, find the existing entry and edit the relevant fields.

### Categories

Use one of the following values for the `category` field:

- `Full Time MSc AI (UK)`
- `Full Time MSc AI (Europe)`
- `Full Time MSc AI (USA)`
- `Part Time MSc AI`
- `Distance Learning / Online MSc AI`

## Building and Testing Locally

The site is static HTML generated by a Node.js build script. You need Node.js installed (any recent version).

```bash
# Install dependencies (none currently, but good practice)
npm install

# Build the site and regenerate the README
node build.js
node generate-readme.js

# Serve locally to preview
npx serve dist
```

Then open `http://localhost:3000` in your browser to check your changes.

## Submitting a Pull Request

1. Fork the repo and create a branch.
2. Edit `data/courses.json`.
3. Run the build steps above and verify the site looks correct.
4. Commit your changes and open a pull request.

The website and README are regenerated automatically on merge, so you only need to edit `data/courses.json`.
